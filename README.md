# JavaScript 学习平台

一个基于 Vite + React 的交互式 JavaScript 学习平台，帮助初学者从零基础系统学习 JavaScript 核心概念。
https://wy5622150.github.io/javascript-algorithm-starter/

## ✨ 功能特点

- **📚 系统化学习路径** - 从基础语法到算法题目，循序渐进
- **💻 实时代码编辑器** - 在线编写和运行 JavaScript 代码
- **🎯 即时反馈** - 实时查看代码执行结果和错误信息
- **📖 详细讲解** - 每个概念都有清晰的说明和示例
- **🎨 类 LeetCode 界面** - 熟悉的三栏布局，提升学习体验
- **✅ 参考答案** - 提供标准解决方案供学习参考

## 🚀 快速开始


### 安装依赖
```bash
npm install
```

### 启动开发服务器
```bash
npm run dev
```

然后在浏览器中打开 http://localhost:5173/

### 构建生产版本
```bash
npm run build
```

## 📖 学习内容

### 基础部分
1. **变量与数据类型** - 学习let、const、var的使用和基本数据类型
2. **引用类型** - 学习数组和对象的使用
3. **条件判断** - 掌握if-else和switch语句
4. **循环** - 学习for、while循环和break/continue控制
5. **函数** - 理解函数声明、箭头函数和作用域
6. **数据类型深入** - 深入理解传值与传址的区别，掌握原始类型和引用类型

### 进阶部分
7. **面向对象编程** - 学习类、对象和继承
8. **递归** - 理解递归思想和应用
9. **栈和队列** - 学习基本数据结构

### 练习部分
10. **循环练习** - 通过15道练习题巩固循环知识，包含基础到进阶的各类题目
11. **函数练习** - 通过15道练习题掌握函数使用，涵盖箭头函数、默认参数等ES6特性
12. **传值与传址练习** - 通过6道练习题深入理解原始类型和引用类型的区别，避免常见错误

## 🛠️ 技术栈

- **React 19** - UI 框架
- **Vite** - 构建工具
- **TailwindCSS** - 样式框架
- **Lucide React** - 图标库
- **CodeMirror 6** - 专业代码编辑器
  - 语法高亮
  - 自动补全
  - 括号匹配
  - 代码折叠
  - 行号显示

## 📝 使用说明

1. **选择课程** - 在左侧列表中选择想要学习的课程
2. **阅读内容** - 中间区域显示课程讲解和示例代码
3. **编写代码** - 在右上方的编辑器中编写代码
4. **运行测试** - 点击"运行"按钮查看结果
5. **查看答案** - 遇到困难时可以查看参考答案

## 🎯 学习建议

1. 按照课程顺序学习，不要跳过基础内容
2. 每学完一个概念，务必在编辑器中实践
3. 尝试修改示例代码，观察不同的结果
4. 先独立思考解决问题，再查看参考答案
5. 多练习，熟能生巧
